Flexibility in Every Detail

Customize Aura for your workflow. Choose global hotkeys, enable toggle mode with a single click instead of holding the key, or change overlay sound themes (from peaceful Zen singing bowls to sci-fi and Rhodes jazz piano).

Uncompromising Privacy

Data privacy is Aura's highest priority. The local mode runs entirely offline on your PC and never transmits your audio recordings over the network. In cloud mode, audio is only sent directly to your chosen provider (Gemini, OpenAI, or Groq) over secure connections. The software contains no hidden tracking systems or telemetry. All settings and API keys are stored locally on your device.

Frequently Asked Questions

Unlike standard Windows dictation, Aura uses intelligent AI models for semantic formatting. It automatically removes filler words ("uh", "um"), cleans up grammar, adds punctuation, and allows you to dictate directly into any active window without losing focus. Additionally, Aura supports fully local offline Whisper speech recognition without sending audio to the internet.

Completely secure. The program has no built-in telemetry or hidden data collection servers. If you use local mode (Whisper Offline), audio data is processed directly on your CPU/GPU and is never transmitted anywhere. When using cloud mode, the audio is sent directly to the providers (Google, OpenAI, Groq) via encrypted connections. API keys are stored locally in the settings file.

For fast offline performance using the "Base" model (~145 MB), any modern 4-core processor and 4 GB of RAM is sufficient. Local Whisper in Aura runs completely autonomously out of the box using your CPU resources, so you do not need to install Python, CUDA libraries, graphics card drivers, or any third-party tools. The application downloads the optimized model file in a single click and is ready to go.
