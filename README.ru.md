Зажмите горячую клавишу, продиктуйте текст, отпустите — Aura распознает речь, уберёт слова-паразиты, расставит пунктуацию и вставит результат в **любое** приложение Windows. Работает с облачным ИИ (Gemini / OpenAI / Groq) или полностью офлайн через локальный Whisper.

Это **бесплатная опенсорс-альтернатива Wispr Flow** — не просто транскрайбер, а голосовой помощник для диктовки *и редактирования*: он приводит сказанное в порядок и может переписать выделенный текст по голосовой команде.

**100% бесплатно и без скрытых платежей** — Aura является проектом с открытым исходным кодом. В ней нет рекламы, платных подписок, ограничений по времени использования или скрытых лимитов.

## Возможности

- **Диктовка по глобальной горячей клавише** — удерживайте для записи или включите режим переключателя (короткое нажатие фиксирует запись); `Esc` отменяет.
- **Два движка**:
  - **Облачный** — Gemini, OpenAI (Whisper + GPT) или Groq (Whisper + Llama), отдельный API-ключ на каждого провайдера.
  - **Локальный** — сайдкар whisper.cpp, 100% офлайн и приватно; модели скачиваются прямо из настроек.
- **ИИ-обработка** — удаление слов-паразитов, пунктуация и грамматика; программа никогда не «отвечает» на надиктованные вопросы — только транскрибирует их.
- **Потоковый ввод** (экспериментальный) — текст печатается по мере диктовки и заменяется финальной версией.
- **История распознаваний** — последние 50 транскрипций с копированием в один клик.
- **Словарь** — свои имена и термины подсказываются распознавателю для точного написания.
- **11 вариантов языка** — автоопределение, по раскладке клавиатуры или фиксированный (ru, en, de, es, fr, it, zh, pt, tr).
- **Голосовые команды пунктуации** (опция) — «запятая», «точка», «новая строка» → знаки препинания.
- **Продуманный оверлей** — живая волна громкости, таймер записи, состояния ошибок и звуковые темы (дзен / родес / sci-fi / классика).
- **Мелочи, которые радуют** — автозапуск с Windows, иконка в трее, защита от второго экземпляра, focus-guard (текст никогда не напечатается в чужое окно).

## Сравнение

Aura находится между минималистичными локальными транскрайберами и платными ИИ-инструментами диктовки. Если нужна просто офлайн-расшифровка — [Handy](https://handy.computer/) отличный кросс-платформенный выбор. Фокус Aura — **слой ИИ-редактирования поверх**, и всё это бесплатно.

| | **Aura** | **Handy** | **Wispr Flow** |
|---|---|---|---|
| Цена | Бесплатно, открытый код | Бесплатно, открытый код | Платно (подписка) |
| Платформы | Windows | Windows / macOS / Linux | Windows / macOS |
| Локальный движок (приватно) | ✅ Whisper / Parakeet | ✅ Whisper / Parakeet | ❌ только облако |
| Облачный движок (без GPU) | ✅ Gemini / OpenAI / Groq | ❌ | ✅ |
| ИИ-обработка (паразиты, грамматика) | ✅ | ➖ только базовая пунктуация | ✅ |
| Редактирование выделенного текста голосом | ✅ | ❌ | ✅ |
| Голосовые команды пунктуации | ✅ | ❌ | ➖ |
| История распознаваний | ✅ | ❌ | ✅ |
| Пользовательский словарь | ✅ | ❌ | ✅ |

> Честные компромиссы: Aura пока только под Windows, а её облачный режим отправляет аудио выбранному провайдеру (локальный — нет). Handy зрелее и кросс-платформенный. Wispr Flow самый вылизанный, но платный и только облачный.

## Установка

Скачайте установщик из [Releases](https://github.com/malashkadev/aura/releases) и запустите. Ознакомиться с интерактивным макетом настроек и посмотреть живую демонстрацию можно на нашем [Официальном сайте](https://aura-beryl-five.vercel.app/).

Для облачного режима нужен API-ключ — бесплатного тарифа [Groq](https://console.groq.com/) более чем достаточно, и он доступен без VPN. Для локального режима просто скачайте модель Whisper из настроек (base — хороший старт).

> **При первом запуске «Система Windows защитила ваш компьютер»?** Установщик пока не подписан цифровым сертификатом (он стоит несколько сотен долларов в год), поэтому SmartScreen предупреждает о новых открытых программах. Нажмите **«Подробнее» → «Выполнить в любом случае»**. Исходный код полностью открыт — при желании можно собрать самостоятельно.

## Использование

| Действие | По умолчанию |
|---|---|
| Начать запись | зажать `Alt+V` |
| Завершить и вставить текст | отпустить клавишу |
| Зафиксировать запись (режим переключателя) | короткое нажатие `Alt+V` |
| Отменить запись | `Esc` |

Горячая клавиша, язык, движок и всё остальное настраивается в окне настроек (иконка в трее → «Открыть настройки»).

## Сборка из исходников

Требования: [Rust](https://rustup.rs/) (stable), [Node.js](https://nodejs.org/) 18+, WebView2 (предустановлен в Windows 11).

```bash
git clone https://github.com/malashkadev/aura.git
cd aura
npm install
npm run dev     # режим разработки
npm run build   # установщик NSIS/MSI в src-tauri/target/release/bundle/
```

Бинарники whisper.cpp лежат в `src-tauri/binaries/`. Чтобы обновить их до свежего релиза whisper.cpp, выполните `python install_whisper.py`.

Тесты:

```bash
cd src-tauri
cargo test
```

## Приватность

- **Локальный режим** ничего никуда не отправляет — аудио обрабатывается на вашем компьютере.
- **Облачный режим** отправляет записанное аудио выбранному провайдеру. Больше ничего не собирается, телеметрии нет.
- Настройки (включая API-ключи) хранятся локально в `%APPDATA%/com.aura.app/settings.json`, история — в `%LOCALAPPDATA%/com.aura.app/history.json`. Известное ограничение: ключи хранятся открытым текстом — миграция на Windows Credential Manager в планах.

## Недавно добавлено

- **Локальный движок Parakeet** — офлайн NVIDIA Parakeet TDT v3 (через [sherpa-onnx](https://github.com/k2-fsa/sherpa-onnx)), работает как резидентный WebSocket-сервер, поэтому модель загружается один раз и распознавание занимает меньше секунды (в отличие от ~12 сек для холодного вызова CLI). Примечание: пользовательский словарь пока не применяется на этом движке, а встроенная пунктуационная модель лучше работает для английского (для русского используйте голосовые команды).
- **VAD (Silero)** — заменяет старый порог тишины на детекцию голоса Silero, чтобы обрезать паузы и убрать галлюцинации.
- **Автообновления** — подписанные релизы со встроенным апдейтером: программа сама проверяет и предлагает установить новые версии.

## Планы развития

В примерном порядке приоритета. Идеи и вклад приветствуются — открывайте issue!

- **Поддержка macOS** — нативный порт (глобальные хоткеи через `CGEventTap`, захват через CoreAudio) уже в основном репозитории и **компилируется в CI**. Для реального запуска ещё нужны macOS-бинарник whisper-сайдкара, `.app`-бандл, поток выдачи Accessibility-прав и тест на живом железе.
- **Пользовательский словарь для Parakeet** — поддержка hotwords через sherpa-onnx (нужно прокидывать файл со словами при старте сервера и перезапускать при изменении).
- **Безопасное хранение ключей** — перенести API-ключи из открытого JSON в системный менеджер учётных данных.
