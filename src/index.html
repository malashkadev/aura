Движок распознавания

Выберите между облачной обработкой высокого качества или полностью автономным локальным распознаванием речи.

Режим ввода текста

Выберите способ отображения наговариваемого текста.

Если выключено: текст вставится целиком только после того, как вы отпустите клавиши (режим версии 1.0). Рекомендуется для стабильной работы с бесплатным API Groq.

Глобальная горячая клавиша

Зажмите выбранную комбинацию для начала записи, отпустите для распознавания и вставки текста.

Настройка облачного API

Укажите ваши API-ключи для авторизации в облачных сервисах Gemini, OpenAI или Groq.

v2.0

Премиальный высокоскоростной клиент голосового ввода. Зажмите горячую клавишу для записи аудио, надиктуйте текст и отпустите для автоматической вставки и пунктуации с помощью искусственного интеллекта.
